## Database Management System ( using bash script )
### Features:

## Database Operations:
- Create Database
- List Database
- Drop database
- Connect To table menu
- Connect Menu

## Connect Menu Operations
  - Create new table 
  - List tables
  - Drop tables
  - Insert into table
  - Update table
  - Delete from table
  - Display Table
  
##  Display Table Operations:

   - Display all records
   - Select Record By Primary-Key
   - Select column by column name

## Delete Fro Table Operations:
 - Delete Record
 - Delete All Records 
### Datatypes:
- String
- Number
## How To Use:
- $ ./Main_menu.sh
